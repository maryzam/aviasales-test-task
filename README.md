## Запуск

Запускаем локально с помощью WebpackDevServer

npm install
npm start

Приложение будет доступно по адресу localhost:9000

## Использовались

- JavaScript, React(v16)
- Moment для форматирования дат
- Jest, enzyme - для тестирования
